# 🚨 Directorio de Iniciativas Digitales - Terremoto Venezuela 2026

Este repositorio centraliza los esfuerzos digitales independientes creados por la comunidad para coordinar la ayuda ante la emergencia nacional. El objetivo es unificar canales de comunicación con los desarrolladores y optimizar el apoyo logístico que está canalizando la empresa Yummy.

### 2. Venezuela Rescate
* **Enlace URL:** `https://venezuelarescate.com/`

* **Líder Técnico / Desarrollador:** Bomberos USB
* **Vías de Contacto Directo:**
