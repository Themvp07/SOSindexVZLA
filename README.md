# 🚨 Directorio de Iniciativas Digitales — Terremoto Venezuela 2026

Este repositorio centraliza los esfuerzos digitales independientes creados por la comunidad para coordinar la ayuda ante la emergencia nacional. El objetivo es ofrecer un punto único de referencia — para personas, organizaciones humanitarias, medios y agentes de IA — sobre qué plataformas existen, qué función cumple cada una, quién las mantiene y cómo contactar a sus equipos de desarrollo.

**Convención de datos:** todo campo sin información verificable en la fuente original se marca como `Por verificar`. No se asumen ni inventan datos.
